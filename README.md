# Ramyandex Desktop Environment

Custom desktop environment built on Linux using Hyprland + Qt6.

## Stack
- **Hyprland** — compositor / window manager
- **Qt6 + QML** — UI / launcher
- **Wayland** — display protocol

## Structure
- `launcher/` — app launcher (Qt6/QML)
- `hypr/` — hyprland config
- `scripts/` — automation scripts
- `sddm/` — login screen
- `wallpapers/` — backgrounds
